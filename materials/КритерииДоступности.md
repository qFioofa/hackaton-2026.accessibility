## 7. Основные проблемы React-приложений (как ФГИС «Спорт»)

| Проблема | Решение |
|----------|---------|
| Динамический контент не озвучивается | Добавить `aria-live` регионы |
| Фокус не перемещается при изменении страницы | Использовать `focus()` после рендера |
| Модальные окна не захватывают фокус | Реализовать trap focus |
| Кастомные компоненты не работают с screen reader | Добавить ARIA-роли и свойства |
| SPA-роутинг не объявляет смену страницы | Использовать `aria-live` или announce route change |

## Как использовать этот чек-лист:

1. **Установите NVDA** (бесплатно: https://nvaccess.org/)
2. **Включите screen reader** перед тестом
3. **Отключите мышь** (или не используйте её) — только клавиатура
4. **Пройдитесь по каждому пункту** и отметьте статус
5. **Запишите найденные ошибки** с описанием и скриншотами
6. **Составьте отчёт** с рекомендациями по исправлению

---

## Типичные ошибки для исправления:

| ⚠️ Ошибка | ✅ Решение |
|-----------|-----------|
| Нет `alt` у изображений | Добавить описательный текст |
| Кнопки без текста | Добавить `aria-label` |
| Поля форм без label | Добавить `<label for="id">` |
| Нет H1 | Добавить главный заголовок |
| Фокус невидим | Добавить CSS outline |
| Цвет — единственный способ различения | Добавить иконки/текст |
| Динамический контент не озвучивается | `aria-live="polite"` |
