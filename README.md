# hackaton-2026.accessibility
Site analize for accessibility in different ways


Логотип «ЕМИР» — аббревиатура, написанная крупными заглавными буквами синего цвета на белом фоне. Расположены горизонтально в одну строку.
Логотип Министерства спорта России — государственный герб в виде двуглавого орла чёрного цвета на белом фоне. В центре щита — силуэт бегущего человека, символизирующий движение и спорт. Орёл держит в лапах скипетр и державу, над головами — три короны. Элементы расположены симметрично, композиция вертикальная.
Логотип «Гостех» — слово, написанное строчными буквами темно-синего цвета на белом фоне. Шрифт современный, без засечек, с плавными округлыми формами. Буквы расположены горизонтально в одну строку, композиция минималистичная.
Логотип «Госуслуги» — слово, написанное строчными буквами. Первая часть «ГОС» выполнена синим цветом, вторая часть «УСЛУГИ» — красным. Шрифт с плавными формами. Буквы расположены горизонтально в одну строку на белом фоне.
Логотип «Спорт — норма жизни» — три строки текста: верхняя строка «СПОРТ» написана заглавными буквами тёмно-серого цвета, рядом с ней — зелёная стрелка вправо. Ниже расположены слова «НОРМА» и «ЖИЗНИ», также заглавными буквами тёмно-серого цвета.  Композиция вертикальная, на белом фоне.
Логотип «Игры будущего» —  графический элемент в виде изогнутой ленты или полосы, выполненной в градиенте от фиолетового внизу до бирюзового вверху. В центре композиции — чёрный треугольник, направленный вправо, напоминающий кнопку «воспроизвести». Фон белый. Композиция динамичная, символизирует движение, технологии и спорт нового формата.
